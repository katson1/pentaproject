#pentaproject

Uma aplicação web, para cadastro de cidades e bairros. Esta aplicação
contem:
- Tela de login
- Tela de cadastro de cidade (nome da cidade, estado e data de fundação);
- Tela de cadastro de bairro (apenas nome),associando a uma cidade;
- Relatório de cidades e bairro (filtro por nome da cidade, data da fundação
  (período inicial e final), nome do bairro.
- Tela de cadastro de usuário.

Foi usado SQLite para o gerenciamento dos dados;
